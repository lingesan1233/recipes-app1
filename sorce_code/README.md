# Recipes CRUD App

A simple REST API to manage recipes using Node.js, Express, and MongoDB (Mongoose).

## Features

- Create, Read, Update, Delete Recipes
- MVC architecture
- MongoDB integration
- Postman API Documentation

## Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- Postman

```bash
npm install
npm run dev
